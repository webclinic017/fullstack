选择合适的高手复制跟随，是件非常重要的事情，交易高手擅长的交易资产和交易风格不尽相同，挑选一个符合收益预期的交易高手跟随和跟随用户的收益息息相关。<%= lang.text('companyZhName')%>的高手都是通过很严格的甄选才可以在排行榜上出现，但是交易高手的历史业绩并不能完全代表未来收益结果，所以风险控制也非常重要，“不要将所有的鸡蛋放在一个篮子里”，选对高手组合对于跟随者是至关重要的。
高手筛选指标
复制人数： 复制人数反映高手的受欢迎程度，往往高手越受欢迎，综合排名也会靠前。
总复制金额： 所有跟随者总共复制该高手的资金。
最低复制资金： 每个高手交易风格不一样，有的高手交易频繁，每笔交易动用的资金偏小，有的高手不轻易下单，看准时机后用更多的资金来进行交易。交易占用的资金占总账户的资金比例越高（占用保证金比例），账户承受风险能力越低。为了保证跟随者的利益，<%= lang.text('companyZhName')%>针对不同风格的交易高手设定了不同的最低复制资金，为了保证跟随的资金安全，建议跟随某个高手时，最低复制金额不要低于要求复制金额。
准确率： 高手平仓的订单中盈利订单的占比。即“平仓盈利单数／总平仓数*100%”获得该高手准确率值。
最大持仓： 高手保证金的最大使用占比。即“持仓占用保证金／余额*100% ”获得高手最大持仓比值。
最大回撤： 高手的最大浮动亏损占比。即“浮动亏损/余额*100%”获得高手最大回撤比值。
投资市场，风险和收益永远都是呈正比例关系，即风险越高，收益越高。最大回撤即代表风险程度，回撤高的高手往往代表风险和收益都比较高。
总收益率： 高手交易累计的收益比率。即“总收益率 = 所有平仓盈亏总和 / 所有入金总和*100%”获得总收益率数值。

关注交易高手过去几个月的交易记录和盈利曲线，来衡量交易高手的风险偏好，不论是稳健型的交易者，还是交易频繁的短线交易者，跟随者可以根据自身需求进行定制。
